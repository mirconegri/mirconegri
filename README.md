## About Me

I'm a first-year Computer Science student at the University of Trento with a focus on databases, programming languages, AI, and big data. Outside of coursework I build tools that solve real problems I encounter — a Telegram bot for dorm laundry bookings, a physical Pomodoro timer with no buttons, a life management app that keeps everything local and offline.

I'm also a volunteer rescuer with A.V.P. and have been active in AGESCI scouting since 2012. I taught IT literacy workshops for seniors at Auser, which shaped how I think about software usability and accessibility.

- Currently learning: algorithms and data structures, functional programming (PolyML), computer architecture (RISC-V)
- Languages I use most right now: Java, C++, Python, JavaScript
- Open to: internships, collaborations, and open source contributions
